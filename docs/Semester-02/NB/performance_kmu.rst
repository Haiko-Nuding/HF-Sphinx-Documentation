Netzwerkperformance & KMU Features
==================================

Performance Probleme
--------------------
* **Duplex Mismatch:** Ein Ende "Full-Duplex", das andere "Half-Duplex" -> Hohe Paketverluste.
* **Broadcast Storms:** Meist durch Loops (Schleifen) ohne Spanning Tree verursacht.
* **Überlastung:** Zu viele Clients an einem zu schwachen Uplink.

KMU/Privat Features
-------------------
* **Gast-Netzwerk:** Isolation der Gäste via VLAN.
* **QoS (Quality of Service):** Priorisierung von VoIP-Telefonie vor Downloads.
* **Captive Portal:** Vorschaltseite für WLAN-Gäste.
