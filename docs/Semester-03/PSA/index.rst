Überblick der Schwerpunkte
--------------------------

* **Woche 1 (Ansible):** Fokus auf **Infrastructure as Code (IaC)**. Einrichtung von Servern, 
  SSH-Key-Management und automatisierte Software-Verteilung.
  
* **Woche 2 (Sockets & Protokolle):** Fokus auf die **Netzwerk-Ebene**. Programmierung von Client-Server-Systemen 
  und Analyse des Datenverkehrs mit Wireshark (TCP-Handshake & Payload).

* **Ausblick Woche 3 (Security):** Erweiterung der Kommunikation um kryptographische Verfahren (Fernet), 
  um die CIA-Triade (Vertraulichkeit) zu vervollständigen.
